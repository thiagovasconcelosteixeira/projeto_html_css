Codigo-fonte

Para escrever um comando dentro do texto, usamos a tag CODE document.getElementById('teste') é escrito em JavaScript

Outra forma é associando a tag CODE com a tag PRE, temos o seguinte modelo. Por exemplo o codigo abaixo em c++ 

 
        
int main() {
    double area , n , raio ;              
        cout << " Qual o raio: ";       
        cin >> raio;       
        cout << " O raio e: " << raio << endl;        
        n = 3.14159;
        area = n* pow(raio, 2);        
cout << "A area e = " << area << endl;    
    return 0;
}
