Principais formatações

Negrito / Destaque

Usando a tag semântica STRONG temos uma frase em Negrito, ou seja, destaca a frase no sentido de dizer que e um trecho forte

Itálico / Ênfase

Para italico usamos a tag semântica EM a tag em signifaca ênfase, ou seja, dar ênfase a frase

Texto Marcado

Para marcar um texto usamos a tag semântica MARK. Usando CSS, podemos mudar a cor da marcação

Texto grando e pequeno

Para um texto grande, semânticamente, em html nao faz mais sentido uar a tag BIG, para ficar dessa forma. Faz mais sentido usar uma classe com css para a formatação do texto.

Porém, para o texto pequeno ainda usa-se a tag ja que semânticamente faz mais sentido

Texto deletado / riscado

Para riscar a frase usamos a tag DEL que significa deletado, uma frase que você lê mas nao precisa considerar

Texto Inserido / Sublinhado

Usando a tag INS damos ênfase a palavra sublinhando dando um efeito de que aquela parte foi preenchida, como por exemplo, um endereço

Texto sobrescrito

Usando a tag SUP para inserir ou escrever coisas do tipo x20

Texto subescrito

Agora usa-se a tag SUB para escrever por exemplo H20
