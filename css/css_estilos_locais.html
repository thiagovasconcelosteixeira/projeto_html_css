Explicando CSS

A tag style é onde entra em si o css, ja que como voce ja deve saber, o CSS é quem vai dar estilo as suas paginas. Dentro das tags style é onde esta contido a configuração do corpo da sua pagina, estilo de fonte, tamanho da fonte, a cor das letras e tudo mais que são adicionado na forma de seletores dentro da tag style, que nada mais é que o comando em html seguindo de chaves, body{} ou h1{}, dessa forma.
