Usando Aruivo css externo

Para melhorar a produtividade, voce pode criar um arquivo .css e nele colocar a configuração de estilo que voce quer para a pagina. Dessa forma, se você tem, por exemplo, 15 pagina, voce nao precisa ficar copiando e colando todo o estilo para todas as paginas, basta usar a tag <link> em todas as paginas e referenciar o arquivo .css criado. No caso do Visual Code, quando voce digitar link, ele ja sugere o link:css, que ja cria a tag para voce.
